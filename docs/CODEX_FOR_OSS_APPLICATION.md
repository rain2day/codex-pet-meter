## Why Does This Repository Qualify? 500 Characters Max

Codex Pet Meter is an MIT-licensed macOS utility for developers who rely on
Codex and Claude Code. It makes local usage state visible around the Codex pet
without modifying Codex.app, requiring Accessibility permission, or sending data
to a maintainer backend. The project is early, but it addresses a real workflow
gap for AI coding-tool users and is maintained as a privacy-conscious OSS tool.
