# Changelog

All notable project changes should be documented here.

The project follows a lightweight versioning model while it is pre-1.0:

- `0.x.0` for visible feature or behavior changes
- `0.x.y` for fixes and documentation-only releases

## 0.3.0 - Unreleased

- Added open-source project documentation for contribution, security, privacy,
  architecture, roadmap, and maintainer workflows.
- Added GitHub issue templates, pull request template, and macOS CI check.
- Added `npm run check` for JavaScript syntax, shell syntax, and Swift
  typechecking.
- Aligned package metadata with the app bundle version.

## 0.2.0

- Added combined Codex and Claude usage support.
- Added provider-specific display modes and color controls.
- Added local usage cache fallback.

## 0.1.0

- Initial public source release.
- Added Swift menu bar app, halo overlay, installer, and preview assets.
